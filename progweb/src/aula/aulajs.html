Exemplo Básico de jQuery

Para usar jQuery, basta incluir a biblioteca e manipular elementos de forma simples. Exemplo:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemplo jQuery</title>
  <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
  <script>
    $(document).ready(function(){
      $("button").click(function(){
        $("p").toggle();
      });
    });
  </script>
</head>
<body>
  <button>Clique para mostrar/ocultar o parágrafo</button>
  <p>Este é um parágrafo que pode ser mostrado ou ocultado.</p>
</body>
</html>

Neste exemplo, o método toggle() é usado para alternar a exibição do parágrafo ao clicar no botão.
