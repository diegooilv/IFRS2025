/* CSS Externo:
   É um arquivo .css separado, que você linka no HTML usando a tag <link>.
   Exemplo:
   <link rel="stylesheet" href="./estilos.css">

/* CSS Interno:
   É um bloco de estilos dentro do próprio HTML, colocado dentro da tag <style> no <head>.
   Exemplo:
   <style>
       body { background-color: #eee; }
   </style>

/* CSS Incorporado (Inline):
   É o estilo aplicado direto no atributo style da tag HTML.
   Exemplo:
   <p style="color: red;">Texto vermelho</p>
*/
