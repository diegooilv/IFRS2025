Simulação de Formulário

Voltar

Utilize este campo para inserir texto simples.

<input type="text" id="text" name="text" placeholder="Digite seu nome">

Utilize este campo para inserir uma senha (os caracteres ficam ocultos).

<input type="password" id="password" name="password" placeholder="Digite sua senha">

Utilize este campo para inserir um endereço de e-mail. O navegador pode validar o formato.

<input type="email" id="email" name="email" placeholder="seuemail@exemplo.com">

Utilize este campo para inserir valores numéricos.

<input type="number" id="number" name="number" placeholder="Digite um número">

Utilize este campo para inserir um número de telefone.

<input type="tel" id="tel" name="tel" placeholder="(00) 00000-0000">

Utilize este campo para inserir URLs. O navegador valida o formato.

<input type="url" id="url" name="url" placeholder="https://exemplo.com">

Utilize este campo para inserir termos de busca.

<input type="search" id="search" name="search" placeholder="Buscar...">

Utilize este campo para selecionar uma data.

<input type="date" id="date" name="date">

Utilize este campo para selecionar um horário.

<input type="time" id="time" name="time">

Utilize este campo para selecionar uma data e hora sem fuso horário.

<input type="datetime-local" id="datetime-local" name="datetime-local">

Utilize este campo para selecionar um mês e ano.

<input type="month" id="month" name="month">

Utilize este campo para selecionar uma semana do ano.

<input type="week" id="week" name="week">

Utilize este campo para selecionar um valor dentro de um intervalo, usando uma barra deslizante.

<input type="range" id="range" name="range" min="0" max="100">

Utilize este campo para selecionar uma cor.

<input type="color" id="color" name="color">

Utilize este campo para fazer upload de arquivos.

<input type="file" id="file" name="file">

Utilize esta caixa para permitir seleção múltipla.

<input type="checkbox" id="checkbox" name="checkbox">

Utilize estes botões para permitir a escolha de apenas uma opção.

<input type="radio" id="radio1" name="radio" value="opcao1"> <label for="radio1">Opção 1</label> <input type="radio" id="radio2" name="radio" value="opcao2"> <label for="radio2">Opção 2</label>

Botão para enviar o formulário.

<input type="submit" value="Enviar">

Botão para limpar todos os campos do formulário.

<input type="reset" value="Limpar">

Botão sem ação padrão, que pode ser usado para outras funções.

<button type="button">Botão Genérico</button>

Botão de envio feito com uma imagem. O atributo src define a URL da imagem exibida.

<input type="image" src="https://via.placeholder.com/150" alt="Imagem" width="150" height="150">
